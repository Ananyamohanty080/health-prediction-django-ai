# Health Prediction Application

## Developed By

Ananya Priyadarsini Mohanty

## Technologies Used

* Python
* Django
* SQLite
* Scikit-learn
* HTML
* CSS

## Features

* Add Patient
* Update Patient
* Delete Patient
* Email Validation
* Date of Birth Validation
* Health Risk Prediction using Machine Learning

## Machine Learning

Decision Tree Classifier is used to predict health risks based on:

* Glucose
* Haemoglobin
* Cholesterol

SQLite Database
